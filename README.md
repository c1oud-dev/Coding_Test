# 🚀 Coding Test Archive

> **"매일 한 걸음씩, 논리적인 사고를 코드로 기록합니다."**
> 단순히 문제를 푸는 것에 그치지 않고, 효율적인 로직을 고민하며 나만의 해결 과정을 차곡차곡 쌓아가는 저장소입니다.
- **Daily Streak**: 매일 최소 한 문제 이상 풀이하며 꾸준함을 유지하려고 노력 중입니다.
- **Thought Process**: 단순 정답 보다는 '**왜 이 알고리즘을 사용했는가**'에 집중하여 논리적인 사고력을 기릅니다.
- **Record & Share**: 해결한 과정은 [Tistory 블로그](https://dev-cloud.tistory.com/)에 상세히 기록하여 지식을 공유하고 복습합니다.
